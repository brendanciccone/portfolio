# Portfolio

My personal portfolio site. Built with Next.js, TypeScript, and Tailwind. Live at [brendanciccone.com](https://www.brendanciccone.com/).

<img width="100%" alt="Screenshot of Brendan Ciccone's portfolio showing a case study" src="https://github.com/user-attachments/assets/2d674430-68ab-4dd1-aa7b-2d2da2abfaef"/>

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Hosted on Vercel

## Run locally

```bash
pnpm install
pnpm dev
```

Then open [localhost:3000](http://localhost:3000).
