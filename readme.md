# Portfolio

My personal portfolio site. Built with Next.js, TypeScript, and Tailwind. Live at [brendanciccone.com](https://www.brendanciccone.com/).

<img width="100%" alt="Screenshot of Brendan Ciccone's portfolio homepage" src="public/og.png"/>

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Hosted on Vercel

## Run locally

```bash
pnpm install
pnpm dev
```

Then open [localhost:3000](http://localhost:3000).
